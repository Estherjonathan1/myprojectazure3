PROVIDER JUSTIFICATION SUMMARY
AWS vs Azure — Business Context Analysis
Team HACK | ProLaunch Careers × Afriment Cohort 19
================================================================

SCENARIO A: Microsoft-Centric Enterprise

For an enterprise organization already operating within the Microsoft ecosystem — using
Windows Server, Active Directory, Microsoft 365, SQL Server, and Azure DevOps — Azure is
the clearly superior choice. The Azure Hybrid Benefit alone eliminates the Windows Server
licensing premium, effectively reducing Windows VM costs to Linux-equivalent pricing and
saving over $40/month per instance compared to standard Windows Pay-As-You-Go rates.

Beyond licensing, Azure provides seamless identity integration through Azure Active
Directory (now Microsoft Entra ID), native compatibility with on-premises Windows workloads
via Azure Arc, and tight integration with tools enterprise teams already use daily. Azure
Reserved Instances offer up to 72% savings for committed workloads, and the Azure portal
provides familiar governance and compliance tooling for regulated industries such as finance
or healthcare. For this profile, the slightly higher base infrastructure cost is more than
offset by licensing savings, reduced migration friction, and operational familiarity.

SCENARIO B: Linux-Based Startup

For a Linux-based startup building cloud-native applications without legacy Microsoft
dependencies, AWS is the more cost-effective and practical choice. At $40.42/month versus
Azure's $45.84/month for equivalent specifications, AWS costs approximately 12% less — a
meaningful difference when every dollar counts in early-stage operations.

AWS also offers the broadest service catalogue, the largest global infrastructure footprint,
the most mature marketplace of third-party integrations, and the deepest community and
documentation support. AWS Savings Plans provide flexible, significant discounts (up to 66%)
without requiring teams to lock into a specific instance type or region, making it ideal for
fast-growing startups whose resource needs evolve rapidly.

RECOMMENDATION SUMMARY:
- Microsoft-centric enterprise → Azure (licensing savings + ecosystem alignment)
- Linux-based startup → AWS (lower base cost + broader ecosystem + flexibility)
